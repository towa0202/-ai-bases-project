AI-Powered Gmail Auto-Reply Agent 🚀

An intelligent Gmail automation tool built with Python, Google Gmail API, and Gemini AI. This agent continuously monitors unread emails, generates context-aware professional replies using Gemini, automatically sends responses, filters newsletters/noreply emails, and manages email status efficiently.

Features
📧 Automatic Gmail inbox monitoring
🤖 AI-generated email replies using Gemini
🔄 Real-time email processing
🚫 Smart filtering for newsletters and no-reply emails
✅ Automatic marking of processed emails as read
🔐 Secure Gmail OAuth authentication
🌐 Customizable social media integration
⚡ Lightweight and easy to deploy
Tech Stack
Python
Gmail API
Google Gemini AI
OAuth 2.0 Authentication
Google API Client Library
